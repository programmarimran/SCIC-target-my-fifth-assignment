Question-2: What is the difference between innerHTML, innerText, and textContent ?

Ans:

*Using innerHTML shows all the elements and text inside the element, including HTML tags. It also shows commented and hidden elements.

*Using innerText shows only the text inside the element.

*Using TextContent shows all the element and content, text and hidden content inside the element .but don't including hidden html tag and commented text and .

Question-3: What is event delegation in the DOM?

Ans:

Event delegation is a method in JavaScript DOM for dynamically handling HTML elements and is the proper way to manage events on a page when interacting with users

Question-4: What is event bubbling in the DOM?

Ans:
Event bubbling is the dynamic handling of HTML elements and works at the right time. It starts first triggered on a child element and then bubbles up to the parent element. Therefore, it is very helpful for element handling in the DOM
